# Chapter 39 — Windows

> *"Split your editor into multiple views for multitasking."*

---

**Difficulty:** ⭐⭐⭐ Intermediate
**Estimated Reading Time:** 15 minutes

---

# Introduction

Windows let you view multiple buffers at the same time in splits.

---

# Window Commands

| Command | Action |
| ------- | ------ |
| `:split` or `:sp` | Split horizontally |
| `:vsplit` or `:vs` | Split vertically |
| `Ctrl+w w` | Jump to next window |
| `Ctrl+w h/j/k/l` | Navigate windows |
| `Ctrl+w =` | Equalize window sizes |
| `Ctrl+w _` | Maximize height |
| `Ctrl+w \|` | Maximize width |
| `:q` | Close current window |
| `:only` | Close all other windows |

---

# Creating Splits from Normal Mode

| Key | Action |
| --- | ------ |
| `Ctrl+w s` | Split horizontally |
| `Ctrl+w v` | Split vertically |
| `Ctrl+w o` | Close other windows (keep only current) |
| `Ctrl+w c` | Close current window |

---

# Resizing Windows

| Command | Action |
| ------- | ------ |
| `Ctrl+w +` | Increase height |
| `Ctrl+w -` | Decrease height |
| `Ctrl+w >` | Increase width |
| `Ctrl+w <` | Decrease width |

---

# Chapter Summary

- `:split` / `Ctrl+w s` — horizontal split
- `:vsplit` / `Ctrl+w v` — vertical split
- `Ctrl+w h/j/k/l` — navigate between windows
- `Ctrl+w =` — equal sizes
- `:only` — close other windows

---

## Exercises

1. Open two files and split vertically.
2. Open a horizontal split with `:split`.
3. Navigate between windows with `Ctrl+w`.
4. Resize windows and equalize them.
