# Introduction

These tips will help you get the most out of Vim.

---

# Tip 1: Use Relative Numbers

```vim
set relativenumber
```

Knowing you're 5 lines away lets you type `5j` instantly.

---

# Tip 2: Learn the Dot Command

The dot command (`.`) repeats the last change.

Master it to eliminate repetitive work.

---

# Tip 3: Use Macros for Repetitive Tasks

Record a macro once.

Replay it hundreds of times.

---

# Tip 4: Use the Quickfix List

```
:copen      →   Open quickfix window
:cnext      →   Next item
:cprev      →   Previous item
```

Useful for compile errors and search results.

---

# Tip 5: Remap Caps Lock

Map Caps Lock to Escape on your OS.

This makes returning to normal mode much faster.

---

# Tip 6: Avoid the Mouse

Force yourself to use keyboard commands.

It's slower at first, but much faster in the long run.

---

# Chapter Summary

- Relative numbers speed up navigation.
- The dot command eliminates repetition.
- Macros automate complex tasks.
- Quickfix handles errors and search results.
- Remap Caps Lock to Escape.
- Avoid the mouse to build muscle memory.

---
