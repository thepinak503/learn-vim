# Introduction

---

# Quick Jumps

```
Ctrl+o      →   Jump back to previous position
Ctrl+i      →   Jump forward
%           →   Matching bracket
*           →   Search word under cursor
#           →   Search backward for word
'           →   Jump to mark
``          →   Jump to previous position
```

---

# Screen Navigation

```
zz          →   Center cursor on screen
zt          →   Cursor to top of screen
zb          →   Cursor to bottom of screen
H           →   Jump to top of screen
M           →   Jump to middle of screen
L           →   Jump to bottom of screen
```

---

# File Navigation

```
gf          →   Go to file under cursor
Ctrl+^      →   Switch to alternate file
:bn/:bp     →   Next/previous buffer
:args       →   List argument files
:prev/:next →   Next/previous argument file
```

---

# Chapter Summary

- `Ctrl+o`/`Ctrl+i` navigate jump history.
- `zz` centers the screen.
- `gf` opens files under cursor.
- `Ctrl+^` toggles between files.

---

## Exercises

1. Practice `*` and `#` for word search.
2. Use `Ctrl+o` and `Ctrl+i` to navigate history.
3. Use `gf` to open a file import.
